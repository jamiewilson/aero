# Examples

- **[kitchen-sink](kitchen-sink/)** — Full demo app with content collections, Nitro API, Alpine.js, and HTMX. Run dev/build from this directory (e.g. `pnpm --dir examples/kitchen-sink dev`); root has no app dev script.
- **[import-bundling](import-bundling/)** — Import and bundling demos (single-bundle, cdn-globals, esm-import-map, cdn-externals, dynamic-import). See [import-bundling/README.md](import-bundling/README.md) and [docs/importing-and-bundling.md](../docs/importing-and-bundling.md).
