Script types & loading behavior

Open the browser console to see each case.

  • script is:inline nested in page content — stays in place, not bundled
  • script props — Vite-bundled client module with build-time data injection
  • script is:inline props — inline script with build-time props
  • Plain script — Vite-bundled client module, hoisted to end of body
  • External script src — third-party URL, left unchanged (not bundled)
  • script is:blocking — moved to head before paint
  • Local script src — aliased path, bundled and rewritten to a hashed asset URL
  • script async src — local module loaded asynchronously
  • script defer src — local module; defer stripped when Aero adds type="module"
  • Public static script src — file from ./public, served as-is
